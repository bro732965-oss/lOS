# lOS

Простая операционная система для офисной работы.

---

## Возможности

- Загрузка с USB-флешки (UEFI)
- Работа с файлами (FAT32)
- Подключение к интернету
- Графический интерфейс
- Работа с GitHub

---

## Как запустить

1. Вставьте флешку в компьютер
2. При включении выберите загрузку с флешки
3. Система запустится автоматически

---

## Команды

| Команда | Что делает |
|---------|------------|
| `HELP` | Показывает подсказки |
| `CLS` | Очищает экран |
| `INFO` | Показывает информацию о системе |
| `LS` | Список файлов |
| `GUI` | Открывает графический интерфейс |
| `GITHUB` | Подключается к GitHub |
| `REBOOT` | Перезагружает компьютер |
| `SHUTDOWN` | Выключает компьютер |

---

## Установка на флешку

1. Отформатируйте флешку в **FAT32**
2. Создайте папку: `EFI/BOOT/`
3. Скопируйте файл как `bootx64.efi`
4. Вставьте флешку и загрузитесь

---

## Требования

- Компьютер с поддержкой UEFI
- Флешка от 4 MB
- (Опционально) Сетевой адаптер для интернета

---

## Поддерживаемые протоколы

- **Файловая система:** FAT32
- **Графика:** GOP (Graphics Output Protocol)
- **Сеть:** TCP/IP, DNS, HTTP/HTTPS
- **API:** sky API
